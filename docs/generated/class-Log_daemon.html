Class Log_daemon

The Log_daemon class is a concrete implementation of the Log:: abstract class which sends messages to syslog daemon on UNIX-like machines. This class uses the syslog protocol: http://www.ietf.org/rfc/rfc3164.txt

Converts a PEAR_LOG_* constant into a syslog LOG_* constant.
